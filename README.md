# ☁️ Cloud DevOps Roadmap

An interactive, visual learning roadmap for Cloud and DevOps technologies. This project provides structured pathways for developers and engineers looking to master modern cloud infrastructure and DevOps practices.

## 🚀 Features

- 📍 Interactive visual roadmap with clickable learning paths
- 📱 Fully responsive design for mobile and desktop
- 🎨 Clean, modern UI with smooth transitions
- 🗺️ Covers AWS, Docker, Kubernetes, CI/CD, and more

## 🛠 Tech Stack

- **JavaScript** – Core interactivity and DOM manipulation
- **HTML5** – Semantic markup and accessibility
- **CSS3** – Flexbox/Grid layouts, animations, and responsive styling

## 📦 Getting Started

```bash
git clone https://github.com/supreeth0008/cloud-devops-roadmap.git
cd cloud-devops-roadmap
# Open index.html in your browser or serve with a live server
```
